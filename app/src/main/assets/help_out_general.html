Allgemeine Einstellungen

Name der Mitarbeiters

Der Name wird standardmäßig beim Erstellen neuer Arbeitszeiten eingetragen, wird beim Versenden im Betreff verwendet und kann auch Teil der Berichtsnummer sein.

Name des Gerätes

Der Gerätename kann als Teil der Berichtsnummer eingebunden werden.

Berichtsnummer

Für die Berichtsnummer kann man mit Hilfe von Platzhaltern ein Muster definieren, so dass nicht nur eine laufende Nummer verwendet wird, sondern z.B. auch eine Jahreszahl beinhaltet sein kann. Die Standardeinstellung (%p-%y-%c) führt zu einer Berichtsnummer: 'Gerätename'-'Jahr der Erstellung'-'laufende Nummer'. Folgende Platzhalter gibt es (achten Sie auf Groß-/Kleinschreibung!):

  • %nc -> laufende Nummer mit n Stellen, z.B. laufende Nummer=17; %6c -> 000017
  • %y -> Jahr der Erstellung
  • %m -> Monat der Erstellung
  • %d -> Tag der Erstellung
  • %e -> Name des Mitarbeits
  • %p -> Name des (Android-) Gerätes
  • %k -> Name des Kunden / Projekts
  • %z -> Projektzusatz

Beachten Sie bitte, dass sich ein Ändern dieser Vorlage auf alle vorhandenen Berichte auswirkt. Dadurch werden sich also die Nummern aller vorhandenen Berichte ändern.

Bildschirmausrichtung

In vielen Fällen, insbesondere bei Geräten ohne Tastatur, ist die Bedienung der App im Querformat nicht wünschenswert. In diesem Fall kann mit der Einstellung 'Bildschirm immer im Hochformat' das automatische Umschalten in das Querformat unterbunden werden.
