Verwendung von SFTP

Eine Liste der Pauschalen, die ODF Vorlage, ein Logo und eine Fusszeile können von einem SFTP Server heruntergeladen werden. SFTP ist ein Protokoll um Dateien über eine gesicherte Verbindung zu übertragen. Es basiert auf SSH.

Dazu muss unter server Adresse der host angegeben, z.B. example.com. Der Standard-port bei SSH ist 22, kann aber je nach Konfiguration des SSH servers anders lauten. Benutzername und Passwort, um sich anzumelden.

Durch Drücken des Testen buttons kann überprüft werden, ob die Verbindung aufgebaut werden kann.

In den Einstellungen, die Dateien herunterladen, muss der Pfad und der Dateiname exakt so angegeben werden, wie er über einen login per SSH verfügbar wäre.
